# About the project

Chess System in Java This project is a fully functional chess application that runs on the console. The goal was to apply advanced Object-Oriented Programming concepts, ensuring that all official rules of the game are respected, from basic movement to complex moves like checkmate.

 

# Technologies used
## Back end
- Java
- Object-Oriented Programming
- Inheritance and polymorphism
- Exception

# How to execute the project

## Prerequisites: Java JDK

```bash
# Clone repository
https://github.com/Cauafernandes-dev/Sistema-Xadrez.git


# Enter the project folder:
cd Sistema-Xadrez

# To execute
java application/Program
```
